Классы и прототипы в JavaScript

Классы и прототипы являются фундаментальными аспектами объектно-ориентированного программирования в JavaScript, позволяющие создавать сложные и структурированные программы.

Классы в JavaScript

Классы в JavaScript представляют собой "синтаксический сахар" над прототипным наследованием и предоставляют более чистый и понятный способ создания объектов и реализации наследования.

        class Animal {
            constructor(name) {
                this.name = name;
            }

            speak() {
                console.log(`${this.name} makes a noise.`);
            }
        }

        class Dog extends Animal {
            constructor(name) {
                super(name); // вызов конструктора родительского класса
            }

            speak() {
                console.log(`${this.name} barks.`);
            }
        }

        let dog = new Dog('Rex');
        dog.speak(); // Rex barks.

Прототипы

Прототипное наследование в JavaScript является механизмом, через который объекты наследуют свойства и методы друг от друга. Это более прямой механизм наследования, по сравнению с классами, который существовал в JavaScript гораздо раньше.

        function Animal(name) {
            this.name = name;
        }

        Animal.prototype.speak = function() {
            console.log(`${this.name} makes a noise.`);
        };

        function Dog(name) {
            Animal.call(this, name); // вызов конструктора Animal
        }

        Dog.prototype = Object.create(Animal.prototype);
        Dog.prototype.constructor = Dog;

        Dog.prototype.speak = function() {
            console.log(`${this.name} barks.`);
        };

        let dog = new Dog('Rex');
        dog.speak(); // Rex barks.

Сравнение классов и прототипов

Классы предоставляют более формальную и понятную структуру для наследования и инкапсуляции, в то время как прототипы предлагают более гибкий подход к наследованию и созданию объектов. Использование зависит от предпочтений разработчика и спецификаций проекта.
