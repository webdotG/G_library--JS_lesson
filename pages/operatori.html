Операторы в JavaScript

Операторы в JavaScript позволяют выполнять действия с переменными и значениями. Они включают арифметические операторы, операторы сравнения и присваивания, логические операторы и другие.

Арифметические операторы

Арифметические операторы используются для выполнения обычных математических операций:

        let x = 10;
        let y = 3;
        console.log(x + y);  // 13
        console.log(x - y);  // 7
        console.log(x * y);  // 30
        console.log(x / y);  // 3.3333
        console.log(x % y);  // 1

Операторы сравнения

Операторы сравнения используются для сравнения двух значений:

        console.log(x == 10);  // true
        console.log(x === 10); // true
        console.log(x != 5);   // true
        console.log(x !== 5);  // true
        console.log(x > 8);    // true
        console.log(x < 8);    // false
        console.log(x >= 10);  // true
        console.log(x <= 10);  // true

Логические операторы

Логические операторы используются для комбинирования булевых выражений:

        console.log(x > 5 && y < 10); // true, оба условия должны быть истинны
        console.log(x > 5 || y > 10); // true, одно из условий должно быть истинно
        console.log(!(x > 5));       // false, инвертирует результат выражения

Дополнительные сведения: Оператор || (ИЛИ) возвращает первое истинное значение или последнее, если все ложные. Оператор && (И) возвращает первое ложное значение или последнее, если все истинные.

Операторы присваивания

Операторы присваивания используются для присвоения значений переменным:

        let a = 10;
        a += 5; // a = a + 5
        console.log(a); // 15

        a -= 5; // a = a - 5
        console.log(a); // 10

        a *= 2; // a = a * 2
        console.log(a); // 20

        a /= 2; // a = a / 2
        console.log(a); // 10

Условный (тернарный) оператор

Условный оператор присваивает значение в зависимости от условия и является сокращением для стандартной конструкции if-else:

        let age = 20;
        let status = (age >= 18) ? "adult" : "minor";
        console.log(status); // "adult"

Объяснение: Если условие age >= 18 истинно, переменной status присваивается значение "adult". В противном случае — "minor".

Деструктуризация в JavaScript

Деструктуризация — это удобный способ извлечения нескольких значений из данных, хранящихся в объектах или массивах, прямо в переменные.

Деструктуризация объектов

Деструктуризация объектов позволяет извлекать значения свойств объекта в переменные с таким же или другим названием.

        const person = {
            name: "Alice",
            age: 25,
            location: "Berlin"
        };

        // Деструктуризация в переменные с тем же названием
        const { name, age, location } = person;
        console.log(name, age, location); // Alice 25 Berlin

        // Деструктуризация в переменные с другим названием
        const { name: userName, age: userAge } = person;
        console.log(userName, userAge); // Alice 25

Деструктуризация массивов

Деструктуризация массивов позволяет извлекать элементы массива в переменные. Это делается по порядку элементов в массиве.

        const numbers = [1, 2, 3, 4, 5];

        // Извлечение первых трёх элементов массива
        const [first, second, third] = numbers;
        console.log(first, second, third); // 1 2 3

        // Пропуск элементов
        const [, , thirdNumber] = numbers;
        console.log(thirdNumber); // 3

Деструктуризация с параметрами функций

Деструктуризация также может быть использована для извлечения значений из объектов, переданных в функцию в качестве параметров.

        function displayUser({ name, age }) {
            console.log(`Name: ${name}, Age: ${age}`);
        }

        displayUser(person); // Name: Alice, Age: 25

Конкатенация строк в JavaScript

Конкатенация строк — это процесс соединения двух или более строк в одну. В JavaScript есть несколько способов сделать это:

Использование оператора `+`

Самый простой и часто используемый метод конкатенации строк — это оператор `+`. Этот оператор позволяет легко соединять строки с переменными или другими строками.

        let hello = "Hello";
        let world = "World";
        let greeting = hello + ", " + world + "!";
        console.log(greeting); // "Hello, World!"

Использование метода `concat()`

Метод `concat()` — это метод объекта String, который также может быть использован для конкатенации строк.

        let string1 = "Hello";
        let string2 = "World";
        let result = string1.concat(", ", string2, "!");
        console.log(result); // "Hello, World!"

Использование шаблонных строк (template strings)

С появлением ES6, шаблонные строки или строковые литералы с подстановками стали предпочтительным способом конкатенации строк. Они позволяют вставлять переменные и выражения прямо в строку, что делает код более читаемым.

        let firstName = "Alice";
        let lastName = "Smith";
        let fullName = `Full name: ${firstName} ${lastName}`;
        console.log(fullName); // "Full name: Alice Smith"

Шаблонные строки особенно полезны, когда вам нужно соединить множество значений или включить сложные выражения в строку.
