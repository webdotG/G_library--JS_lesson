JavaScript в браузере

Браузер — это программа для отображения веб-страниц. Он обладает следующими основными компонентами для работы с JavaScript:

Основные компоненты браузера

  • Объектная модель документа (DOM)

    DOM позволяет JavaScript взаимодействовать с содержимым веб-страницы, изменяя его структуру, стиль или содержание.

  • Объектная модель браузера (BOM)

    BOM предоставляет объекты для работы с браузером, включая управление окнами, местоположением, историей и другими функциями, не связанными напрямую с документом.

  • JavaScript Engine

    Движок JavaScript, такой как V8 в Google Chrome, интерпретирует и выполняет JavaScript код.

Способы подключения JavaScript к веб-странице

  • Внутренний JavaScript

    JavaScript код размещается непосредственно в HTML документе внутри тегов <script>.

    <script>
console.log('Привет, мир!');
</script>

  • Внешний JavaScript

    JavaScript код размещается в отдельном файле и подключается к HTML документу с помощью атрибута src тега <script>.

    <script src="script.js"></script>

Загрузка и выполнение JavaScript

При загрузке страницы браузер обрабатывает HTML, построение DOM и последовательно загружает и выполняет скрипты. Атрибуты async и defer управляют порядком выполнения внешних скриптов:

  • defer

    Скрипты с атрибутом defer загружаются в фоне и выполняются после полной загрузки HTML, но перед событием DOMContentLoaded.

  • async

    Скрипты с атрибутом async загружаются независимо от загрузки страницы и выполняются как только будут загружены, не блокируя рендеринг страницы.

Построение DOM и CSSOM

После загрузки HTML-кода браузер строит DOM (структура документа) и CSSOM (стили), объединяет их для создания рендер-дерева, которое затем используется для отрисовки страницы на экране.

События в JavaScript

JavaScript реагирует на пользовательские действия через события, такие как клики, наведение мыши или нажатия клавиш. Обработчики событий можно добавлять напрямую к элементам:

document.getElementById('myButton').onclick = function() {
    alert('Кнопка нажата!');
};
