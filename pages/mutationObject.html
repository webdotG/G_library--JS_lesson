Мутация объектов в JavaScript

Мутация объектов относится к изменениям, которые можно внести в объект после его создания. В JavaScript объекты являются ссылочными типами, что означает, что изменения в объекте могут влиять на все ссылки на этот объект.

Изменение существующих свойств

Модификация значения существующего свойства объекта является одним из наиболее частых примеров мутации.

        let user = {
            name: "Alice",
            age: 25
        };

        // Изменение свойства
        user.age = 26;
        console.log(user.age); // 26

Добавление новых свойств

Добавление новых свойств также изменяет структуру объекта.

        // Добавление нового свойства
        user.location = "Russia";
        console.log(user.location); // "Russia"

Удаление свойств

Удаление свойств из объекта — ещё один вид мутации.

        // Удаление свойства
        delete user.age;
        console.log(user.age); // undefined

Предотвращение мутаций

JavaScript предоставляет несколько методов для предотвращения или ограничения мутаций объектов.

  • Object.freeze() - Запрещает добавление новых свойств, удаление старых свойств и изменение существующих свойств объекта.
  • Object.seal() - Запрещает добавление новых свойств и удаление существующих, но позволяет изменять существующие свойства.
  • Object.preventExtensions() - Запрещает добавление новых свойств, но позволяет изменять и удалять существующие свойства.
        Object.freeze(user);
        user.age = 30; // Не работает, если объект заморожен
        console.log(user.age); // 26
