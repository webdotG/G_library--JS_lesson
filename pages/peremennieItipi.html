Переменные и типы данных в JavaScript

Переменные

В JavaScript переменные могут быть объявлены с помощью трех ключевых слов: var, let и const.

  • var – традиционный способ объявления переменных, доступен во всех версиях JavaScript.
  • let – современный способ объявления переменных с блочной областью видимости.
  • const – также имеет блочную область видимости, но используется для объявления констант, значения которых не могут быть изменены после установки.

Примеры объявления переменных

        var name = 'Alice';
        let age = 25;
        const birthday = 'April 12';

Типы данных

JavaScript — язык с динамической типизацией. Все типы данных могут быть разделены на две категории: примитивы и объекты.

Примитивные типы

  • String – строки, например "Hello, world!"
  • Number – числа, могут быть целыми или с плавающей точкой.
  • Boolean – логический тип данных, имеет всего два значения: true и false.
  • undefined – тип для неопределенных переменных.
  • null – тип для объектов, у которых нет значения.
  • Symbol – используется для создания уникальных идентификаторов.
  • BigInt – для работы с числами, размер которых превышает Number.MAX_SAFE_INTEGER.

Тип объект

Объекты в JavaScript — это структуры данных, которые могут содержать набор пар ключ-значение. Объекты могут быть использованы для создания массивов, функций, и других сложных структур.

Примеры использования типов данных

        let message = "Hello, world!";  // String
        let number = 42;               // Number
        let isJavaScriptFun = true;    // Boolean
        let user = {                   // Object
            name: "Alice",
            age: 25
        };
        let bigNumber = 123456789012345678901234567890n;  // BigInt
