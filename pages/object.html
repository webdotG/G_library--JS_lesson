Объекты в JavaScript

Объекты — это коллекции пар ключ-значение, где ключами являются строки, а значениями могут быть как примитивы, так и другие объекты. Объекты в JavaScript могут быть использованы для организации данных и поведения в структурированный формат.

Создание объектов

Существует несколько способов создать объект в JavaScript:

  • Литеральная нотация
  • Конструктор Object()
  • Конструкторы, определённые пользователем
  • Метод Object.create()

Примеры создания объектов

        // Литеральная нотация
        let person = {
            name: "Alice",
            age: 25
        };

        // Конструктор Object()
        let person2 = new Object();
        person2.name = "Bob";
        person2.age = 30;

        // Конструктор, определённый пользователем
        function Person(name, age) {
            this.name = name;
            this.age = age;
        }
        let person3 = new Person("Carol", 27);

        // Object.create()
        let person4 = Object.create(person);
        person4.name = "Dave";

Работа с объектами

Доступ к свойствам объекта можно получить двумя способами:

  • Точечная нотация: person.name
  • Квадратные скобки: person["name"]

Добавление и удаление свойств

        // Добавление свойства
        person.city = "New York";

        // Удаление свойства
        delete person.age;

Прототипы и наследование

Каждый объект в JavaScript имеет "прототип", от которого он наследует свойства и методы. Прототипы являются основой для наследования в JavaScript.

Пример наследования

        // У объекта person4 уже есть свойства name и age, наследованные от person
        console.log(person4.name); // выводит "Dave"
        console.log(person4.age);  // выводит 25, унаследовано от person
