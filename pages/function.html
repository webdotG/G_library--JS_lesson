Функции в JavaScript

Функции — это основные "строительные блоки" программ на JavaScript. Они позволяют коду быть вызванным многократно, сохраняя при этом чистоту и порядок в структуре кода.

Определение функции

Существует несколько способов определения функций в JavaScript:

  • Объявление функции (Function Declaration)
  • Функциональное выражение (Function Expression)
  • Стрелочные функции (Arrow Functions)

Примеры определения функций

        // Объявление функции
        function greet(name) {
            return "Hello " + name + "!";
        }

        // Функциональное выражение
        const square = function(number) {
            return number * number;
        };

        // Стрелочная функция
        const getSum = (a, b) => a + b;

Вызов функции

Функцию можно вызвать, используя её имя, за которым следуют круглые скобки с аргументами, если они необходимы.

        console.log(greet("Alice")); // "Hello Alice!"
        console.log(square(5));      // 25
        console.log(getSum(3, 4));   // 7

Параметры и аргументы

Функции могут принимать параметры, которые представляют собой переменные, используемые в определении функции. При вызове функции, ей передаются аргументы, которые являются значениями этих переменных.

Значения по умолчанию

В ES6 и новее можно задавать значения по умолчанию для параметров функции.

        function greet(name, greeting = "Hello") {
            return greeting + " " + name + "!";
        }

        console.log(greet("Alice"));              // "Hello Alice!"
        console.log(greet("Bob", "Good morning")); // "Good morning Bob!"

Возврат значения

Функции могут возвращать значения. Если функция не возвращает значение, по умолчанию она возвращает undefined.

        function getArea(width, height) {
            return width * height;
        }

        let area = getArea(5, 3);
        console.log(area); // 15
