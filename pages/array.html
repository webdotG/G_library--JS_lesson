Массивы в JavaScript

Массивы в JavaScript — это высокоуровневые списочные структуры, которые используются для хранения упорядоченных коллекций данных.

Создание массивов

Массивы можно создавать с помощью литеральной нотации или конструктора Array.

        // Литеральная нотация
        let fruits = ["Apple", "Banana", "Cherry"];

        // Конструктор Array
        let numbers = new Array(1, 2, 3, 4, 5);

Доступ к элементам и изменение массивов

Элементы массива доступны по их индексу. Индексация начинается с нуля.

        console.log(fruits[0]); // Apple
        fruits[1] = "Blackberry";
        console.log(fruits); // ["Apple", "Blackberry", "Cherry"]

Методы работы с массивами

JavaScript предоставляет множество методов для работы с массивами, включая добавление, удаление элементов и многое другое.

        // Добавление элемента в конец массива
        fruits.push("Durian");
        console.log(fruits);

        // Удаление последнего элемента массива
        fruits.pop();
        console.log(fruits);

        // Добавление элемента в начало массива
        fruits.unshift("Strawberry");
        console.log(fruits);

        // Удаление первого элемента массива
        fruits.shift();
        console.log(fruits);

Итерация по массивам

Для итерации по массивам можно использовать различные циклы и методы массивов.

        // Цикл for
        for (let i = 0; i < fruits.length; i++) {
            console.log(fruits[i]);
        }

        // Метод forEach
        fruits.forEach(function(item, index, array) {
            console.log(item, index);
        });

Другие полезные методы массивов

Массивы в JavaScript включают методы для фильтрации, преобразования элементов, сортировки, и многого другого.

        // Метод map
        let lengths = fruits.map(function(item) {
            return item.length;
        });
        console.log(lengths);

        // Метод filter
        let longFruits = fruits.filter(function(item) {
            return item.length > 6;
        });
        console.log(longFruits);

        // Метод sort
        let sortedFruits = fruits.sort();
        console.log(sortedFruits);
